# OpenClaw Evo Skill

Monitor and control the OpenClaw self-evolution system directly from chat.

## Commands

### /evo status
Show current hub status: running state, total cycles, deployed skills, active experiments.

### /evo stats
Show performance metrics: overall score, top tools by success rate, failure patterns.

### /evo cycles
List recent evolution cycles with their outcomes.

### /evo skills
List all proposed and deployed skills with confidence scores.

### /evo experiments
Show active and completed A/B experiments with results.

### /evo trigger
Manually trigger an evolution cycle (requires hub running).

### /evo approve <skill-id>
Approve a proposed skill for experimentation.

### /evo reject <skill-id>
Reject a proposed skill.

### /evo logs
Show recent improvement log entries.

### /evo help
Show this help message.
